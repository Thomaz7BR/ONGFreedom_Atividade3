# ONGFreedom_Atividade3


# Objetivo

Esta terceira entrega tem como foco a implementação de JavaScript avançado, tornando a aplicação web da ONG Freedom dinâmica e interativa.
Foram aplicados conceitos de manipulação do DOM, eventos, armazenamento local (LocalStorage) e sistema SPA (Single Page Application), simulando o comportamento de uma aplicação real de uma ONG.

# Funcionalidades Implementadas

 Sistema SPA (Single Page Application)
O site utiliza JavaScript modular para carregar páginas dinamicamente sem recarregar o navegador.
Os templates (index, projetos, cadastro) são gerenciados pelo arquivo spa.js, que atualiza o conteúdo do <main> com base no link clicado no menu.

#  Sistema de Templates em JavaScript
Cada seção (como index, projetos e cadastro) é definida em arquivos JavaScript separados, permitindo a troca dinâmica de conteúdo.

#  Validação de Formulário (Cadastro)
O formulário realiza verificação de consistência dos dados (campos obrigatórios, formato de e-mail, etc.).
Em caso de erro, o usuário recebe mensagens de aviso visuais e textuais.

#  Armazenamento Local (LocalStorage)
As informações preenchidas nos formulários são salvas localmente no navegador, permitindo recuperação e persistência dos dados.

#  Menu Interativo (Hambúrguer Menu)
O menu se adapta para dispositivos móveis, sendo aberto e fechado por meio de evento JavaScript.
